# 贡献指南

感谢参与 AeroHalo360。安全相关修改必须同时更新实现、中文说明、配置和测试，不能只改变默认参数。

1. 从独立分支提交小而清晰的改动。
2. 不提交真实 SN、IP、串口硬件 ID、用户名、rosbag、密钥或个人配置。
3. 在 ROS 2 Humble 环境运行构建、全部测试和 `nano_deploy/08_static_checks.py`。
4. 涉及 TF、Watchdog、MAVLink、过滤或 ArduPilot 行为时，说明失败语义和新增测试。
5. 真实硬件结论必须附平台、固件、参数、时间和可复现步骤；未验证项明确标为待 HIL。

漏洞或可能导致飞行危险的问题不要只在公开讨论中给出可直接利用的细节，请按 SECURITY.md 报告。
